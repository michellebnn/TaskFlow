# 📋 Requisitos do Sistema

## Nome do Projeto

TaskFlow

---

## Objetivo

Desenvolver uma aplicação web para gerenciamento de projetos e tarefas, permitindo que equipes organizem atividades, acompanhem o progresso e colaborem de forma eficiente.

---

# Tecnologias

## Front-end

- Angular
- TypeScript
- Angular Material
- SCSS

## Back-end

- NestJS
- TypeScript
- Prisma ORM
- JWT

## Banco de Dados

- PostgreSQL

---

# Funcionalidades

## Autenticação

- Cadastro de usuários
- Login
- Logout

## Usuários

- Criar usuário
- Editar usuário
- Excluir usuário

## Projetos

- Criar projeto
- Editar projeto
- Excluir projeto
- Listar projetos

## Tarefas

- Criar tarefa
- Editar tarefa
- Excluir tarefa
- Alterar status
- Definir prioridade
- Definir responsável
- Definir prazo

## Comentários

- Adicionar comentários às tarefas

## Dashboard

- Total de projetos
- Total de tarefas
- Tarefas pendentes
- Tarefas concluídas

# Perfis

## Administrador

- Gerencia usuários
- Gerencia projetos
- Gerencia tarefas
